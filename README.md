# ChipKart E-commerce 🛒

A full-featured e-commerce web application built with **Node.js** and **EJS**, offering a seamless shopping experience with server-side rendering and complete commerce functionality.

---

## 📌 Overview

ChipKart is a server-rendered e-commerce platform where users can browse products, manage carts, place orders, and make payments using COD or Razorpay.
It also includes a powerful admin panel to manage products, users, orders, and promotional features efficiently.

---

## 🔥 Key Highlights

* ⚡ Server-side rendering using EJS for fast and SEO-friendly pages
* 🛒 Complete e-commerce workflow (browse → cart → checkout → order)
* 💳 Integrated Razorpay + Cash on Delivery (COD)
* 🎟 Coupon & referral system for discounts and user engagement
* ❤️ Wishlist functionality for saving products
* ⭐ Product reviews and ratings system
* 📦 Order tracking and lifecycle management
* 🛠 Admin panel for full system control

## 🚀 Getting Started

### Prerequisites

* Node.js (v14+)
* MongoDB

---

### 🔧 Installation

```bash
git clone https://github.com/anandak19/Chipkart-computers.git
cd Chipkart-computers
npm install
```

---

### ▶️ Run the Application

```bash
npm start
```

> Uses nodemon for development:

```bash
nodemon server.js
```

---

## 📡 Core Functional Flow

1. User browses products
2. Adds items to cart or wishlist
3. Applies coupon / referral benefits
4. Selects delivery address
5. Chooses payment method
6. Places order
7. Tracks order status
8. Adds review and rating
